# Publicar EVENTIA

La carpeta mas limpia para subir el sitio es:

`C:\Users\benja\OneDrive\Documentos\EVENTIA-web`

## Opcion 1: Netlify

1. Abre Netlify en tu navegador.
2. Inicia sesion o crea una cuenta.
3. Busca la opcion para subir un sitio manualmente.
4. Arrastra la carpeta `EVENTIA-web` o su contenido.
5. Espera a que termine la carga.
6. Netlify te dara un enlace publico para compartir.

## Opcion 2: GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube el contenido de la carpeta `EVENTIA-web`.
3. Activa GitHub Pages desde la configuracion del repositorio.
4. Selecciona la rama principal y la carpeta raiz.
5. Guarda los cambios y espera a que se publique.

## Archivos importantes

- `index.html`
- `eventia-site.css`
- `eventia-site.js`
- `eventia-logo-transparent.png`
- `eventia-logo.png.png`

## Nota

Si haces cambios al sitio en `Documentos`, vuelve a copiar los archivos actualizados dentro de `EVENTIA-web` antes de subirlos otra vez.
